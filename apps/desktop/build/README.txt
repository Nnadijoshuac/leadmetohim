Place your icon files here:
  - icon.ico  (Windows icon, 256x256 recommended)
  - icon.png  (PNG version for cross-platform)

You can generate icon.ico from a PNG using:
  - Online: https://convertio.co/png-ico/
  - CLI: npx electron-icon-builder --input=icon.png --output=./build

The icon should be a minimalist gold cross or scripture-related symbol
on a dark background that matches the LeadMeToHim brand.
